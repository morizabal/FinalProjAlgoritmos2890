Este directorio contiene tres ejercicios realizados para la completar el proyecto final del curso de **Algoritmos 2890**.

# Numeros amigos.cpp
**Descripcion**
Pide dos números positivos

Calcula sus divisores propios

Suma esos divisores

Compara si cumplen la condición para ser amigos

Muestra el resultado

# Funcion de parametros de dos numeros.cpp

**Descripcion**

Pide dos números enteros positivos.

Identifica cuál es el mayor y cuál es el menor.

Resta repetidamente el menor al mayor hasta que ya no se pueda seguir restando.

El valor que queda es el resto de la división.

# Funcion logica de dos argumentos enteros.cpp

**Descripcion**

Pide dos números enteros.

Revisa si a divide a b o si b divide a a.

Si alguno de los dos divide exactamente al otro → muestra "Uno divide al otro".

Si no → muestra "Ninguno divide al otro"
