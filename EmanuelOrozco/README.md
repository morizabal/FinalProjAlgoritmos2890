## ClasificarEnteroEnCategorias.cpp
Descripción: El usuario ingresa un Número y el programa clasifica si el número es menor a 0, mayor a 100 o está entre 0 y 100.
Ejemplo: 
Entrada -> 30
Salida -> 0 <= x <= 100
 ---

 ## CalcularSalarioDeTrabajadores.cpp
 Descripción: Se desea calcular el salario neto semanal de los trabajadores de una empresa de acuerdo a las siguientes normas:
Horas semana les trabajadas < = 38, a una tasa dada.
Horas extras (38 o más), a una tasa 50 por 100 superior a la ordinaria.
Impuestos 0 por 100, si el salario bruto es menor o igual a 300 euros.
Impuestos 10 por 100, si el salario bruto es mayor de euros.
Entrada esperada: horas = 45, tasa = 10
Salida esperada: Salario bruto 380, Salario neto 342
---

## AlgoritmoDeEuclides.cpp
Descripción:  Es un programa que implementa el algoritmo de euclides para encontrar el máximo común divisor entre dos numeros enteros y positivos
Entrada esperada: a = 48, b = 18
Salida esperada: MDC = 6
