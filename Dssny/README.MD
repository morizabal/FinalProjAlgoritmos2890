# Calculo de MCD y MCM
Descripción 
Este programa solicita dos números enteros e identifica el Máximo Común Divisor (MCD) utilizando el algoritmo de Euclides. Luego calcula el Mínimo Común Múltiplo (MCM) aplicando la fórmula (a * b) / MCD.

Ejemplo de uso:
Entrada:
12
18
Salida:
MCD: 6
MCM: 36

#Numros pares entre 1 y 50
Descripción
El programa recorre los números del 1 al 50 y muestra únicamente aquellos que sean pares, utilizando la condición i % 2 == 0 dentro de un ciclo for.

Ejemplo de salida:
2
4
6
...
50

#Funcion que devuelve P o N
Descripción
Este programa implementa una función que recibe un número entero y devuelve la letra ‘P’ si el número es positivo, o ‘N’ si es cero o negativo. Luego, el programa principal muestra el resultado en pantalla.

Ejemplo de uso:
Entrada:
-3
Salida:
N

Entrada:
10
Salida:
P
