2. Programa: serie_formula
En lugar de sumar término por término, usa una fórmula cerrada para obtener el resultado de forma directa y eficiente.

3. Programa: suma
Realiza la suma de los números desde 1 hasta n utilizando un ciclo for. El usuario ingresa un número positivo y el programa calcula la suma acumulada de manera iterativa.
